CCMS

基于MVC模式的电信资费管理系统

View on GitHub Download .zip Download .tar.gz

功能需求：

实现前台访客页面与后台管理，总体需求模块如下：
* 登陆管理：实现登陆后方可使用该应用程序。
* 用户管理：增、删、改、查用户信息。
* 管理员管理：增、删、改、查管理员信息。
* 资费管理：增、删、改、查资费信息。
* 用户自服务管理：用户修改密码，查询账单。
* 数据采集模块：实现数据采集。
* 账单管理：按月生成用户计费账单。
* 帐务管理：按月生成帐务月、年统计报表。并生成统计图形和打印报表。

性能需求：

  • 站点中网页的外观设计和内容应相互独立。
  • 站点要有统一的风格和布局：使用母版页、内容页、主题。
  • 站点要为用户提供方便的站点导航。
